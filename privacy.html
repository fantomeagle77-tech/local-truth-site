1. О чём эта политика

Эта политика описывает, какие данные может собирать и обрабатывать приложение «Что по району — Где реально норм», для чего они используются, как они защищаются, как долго хранятся и как пользователь может запросить удаление данных.

2. Какие данные мы можем собирать

  • Email адрес — при регистрации и входе в аккаунт.
  • Идентификатор пользователя — для привязки профиля, публикаций, реакций и комментариев.
  • Пользовательский контент — отзывы, комментарии, жалобы, реакции, текст публикаций.
  • Фотографии — если пользователь сам загружает фото к публикациям.

3. Какие данные мы не собираем как обязательные

  • Геолокация в фоне.
  • Контакты устройства.
  • Номер телефона, если пользователь сам его не передаёт отдельно.

4. Для чего используются данные

  • Для создания и обслуживания аккаунта пользователя.
  • Для отображения публикаций, комментариев, реакций и загруженных фотографий.
  • Для синхронизации данных между устройствами.
  • Для модерации спорного контента и обработки жалоб.
  • Для обеспечения стабильной и безопасной работы приложения.

Мы не продаём персональные данные пользователей. Данные могут обрабатываться инфраструктурными сервисами, которые используются для работы приложения, хранения данных, аутентификации и загрузки пользовательского контента.

Такие сервисы используются только для предоставления функциональности приложения, хранения и обработки данных от нашего имени.

  • Доступ к данным ограничивается техническими и организационными мерами безопасности.
  • Мы стремимся хранить только тот объём данных, который действительно нужен для работы приложения.

8. Срок хранения данных

Данные хранятся до тех пор, пока это необходимо для работы аккаунта и функций приложения, если более длительное хранение не требуется по причинам безопасности, предотвращения злоупотреблений, разрешения споров или выполнения юридических обязательств.
